Ainda no tópico *Visualização de Dados*, da formação [Data Science](https://www.alura.com.br/formacao-data-science), oferecido pela [Alura](https://www.alura.com.br/), passamos o foco para aprender a criar gráficos e visualizações de dados com a biblioteca Matplotlib. Desta vez, o curso foi ministrado pelo [Luis Meazzini](https://www.linkedin.com/in/lmeazzini/) e aprendi:

- conceitos sobre o funcionamento de figuras e eixos, 
- a personalizar as visualizações com diferentes cores, estilos e anotações, 
- e criar visualizações para diversos tipos de dados, como séries temporais e dados categóricos.
